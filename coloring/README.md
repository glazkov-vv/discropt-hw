# Попытка 1. Генетический алгоритм. 

Ищем ответ в виде оптимальной перестановки, после чего красим жадно.

Исходно генерируем семейство случайных перестановок, после этого на каждой итерации разбиваем популяцию по парам, каждая из которых производит перестановку-потомка (естественным образом, упоядочиваем элементы по сумме позиций в двух перестановках). Прореживаем популяцию до ее исходного размера, причем половину элементов убиваем с конца, а половину - случайным образом (элита иммунна от убийств).

Такой подход позволяет пройти все легкие тесты, кроме одного.

Чтобы продвинуться дальше, была добавлена эвристика, которая вместо разбиения на пары случайным образом пытается максимизировать "непохожесть" элементов в паре (жадно набираем к элементу максимально непохожий)

Такой подход позволил пройти все легкие и один сложный тест.
