## Как запускать

Данные о тестах и порогах указываются в файле `tests.txt`

Для запуска тестов используется скрипт `build_and_run.sh`. Результат выводится на экран и сохраняется в файле `report.txt`

## Отчет о работе

Реализован жадный алгоритм, оценивающий для каждого множества величину "оптимальности", вычисляемую как отношение ранее не покрытых обхектов, покрываемым им, к стоимости этого множества и берущий объект с максимальной оптимальностью.
