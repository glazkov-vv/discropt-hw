# Как запускать

Данные о тестах и порогах указываются в файле `tests.txt`

Для запуска тестов используется скрипт `build_and_run.sh`. Результат выводится на экран и сохраняется в файле `report.txt`

# Отчет о работе

## Попытка 1. Жадный алгоритм. 

Жадный алгоритм, сортирующий объекты по убыванию "оптимальности", рассчитанной как отношение стоимости предмета к весу и берущий все объекты при возможности. 

Решение проходит все простые тесты, кроме одного. 

## Попытка 2. Перебор + жадный алгоритм.

Перебираем несколько предметов, которые точно берем, остальные добираем предыдущим жадным алгоритмом. 

С увеличением глубины перебора получаем решение, отношение стоимости которого к оптимальному сколь угодно близкое к единице. 

На практике выбираем глубину перебора адаптивно: фиксируем "бюджет" времени исполнения исходя из ограничения по времени, и оцениваем число операций как N^(depth + 1) * log(N). Выбираем максимальную подъодящую под бюджет глубину. 

Решение проходит все простые и сложные тесты.
